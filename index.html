📽️ Cas pratique : Présentation de mon métier
🎯
Objectif : Créer une présentation PowerPoint de 5 diapositives pour présenter votre métier à un employeur ou jury.
B1 PowerPoint 2019 ⏱ 45 minutes 10 étapes
✓
Ouvrir PowerPoint 2019
🖱️ Double-cliquez sur l'icône PowerPoint sur le bureau (ou cherchez "PowerPoint" dans le menu Démarrer) → Cliquez sur "Nouvelle présentation vierge"
✓
Choisir un thème professionnel
🎨 Onglet Conception → Choisissez un thème sobre et professionnel → Cliquez dessus pour l'appliquer
✓
Diapositive de titre
📝 Cliquez dans la zone "Titre" → Tapez votre prénom, nom et votre métier. Zone sous-titre : tapez la date et votre formation.
✓
Ajouter une 2e diapositive
➕ Onglet Accueil → bouton "Nouvelle diapositive" → Choisissez "Titre et contenu" → Titre : "Mon métier" → Listez 3 à 5 tâches principales
5
⚠️ Insérer une image depuis VOTRE ordinateur
IMPORTANT Onglet Insertion → "Images" → "Cet appareil" (PAS d'images en ligne !) → Naviguez vers votre image → Cliquez sur "Insérer"
6
Ajouter les diapositives 3, 4, 5
Répétez la démarche : "Ma formation", "Mes compétences", "Conclusion / Contact"
7
💾 Enregistrer (OBLIGATOIRE)
Fichier → Enregistrer sous → Dossier Documents → Nom du fichier : NomRéférente_Groupe3_PPT_Metier → Cliquez sur "Enregistrer"
💾
Rappel enregistrement : Fichier → Enregistrer sous → Documents → NomRéférente_Groupe3_PPT_Metier.pptx
📄 Cas pratique : Lettre de candidature
🎯
Objectif : Rédiger une lettre de candidature professionnelle avec Word 2019.
1
Ouvrir Word 2019 — Nouveau document vierge
🖱️ Menu Démarrer → "Word 2019" → Cliquez sur "Document vierge"
2
Configurer la mise en page
Onglet Mise en page → Marges → Normales (2,5 cm). Police : Calibri 12pt. Interligne : 1,5
3
Saisir vos coordonnées en haut à droite
Prénom Nom → Adresse → Téléphone → E-mail. Alignez à droite (Ctrl+R)
4
⚠️ Enregistrer le document
Ctrl+S → Documents → NomRéférente_Groupe_Lettre.docx
📊 Cas pratique : Planning hebdomadaire
🎯
Objectif : Créer un tableau de planning sur Excel 2019 avec les jours de la semaine et vos tâches.
1
Ouvrir Excel 2019
Menu Démarrer → "Excel 2019" → Cliquez sur "Classeur vierge"
2
Saisir les jours de la semaine
Cellule B1 : tapez "Lundi" → C1 : "Mardi" → D1 : "Mercredi" → E1 : "Jeudi" → F1 : "Vendredi"
3
⚠️ Enregistrer
Ctrl+S → Documents → NomRéférente_Groupe_Planning.xlsx
